## Try it yourself!

### Run

1. Start containers with Docker Compose:

```shell
$ cd src
$ docker-compose up
```

2. Explore the API on http://localhost:880/api/docs

### Cleanup

Stop containers with Docker Compose:

```shell
$ docker-compose down
```
